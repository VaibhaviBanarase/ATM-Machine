
# ATM Machine

A Java-based **ATM Management System** developed using **Java Swing** and **MySQL** in **NetBeans IDE**. This project simulates the basic functionalities of an Automated Teller Machine through a user-friendly graphical interface.

## Features

* Secure User Login
* Deposit Money
* Withdraw Money
* Check Account Balance

## Technologies Used

* Java
* Java Swing (GUI)
* MySQL (`atm_db`)
* NetBeans IDE

## How to Run

1. Open the project in NetBeans IDE.
2. Import the MySQL database (`atm_db`).
3. Update the database connection details if required.
4. Build and run the project.
5. Log in and use the available ATM features.
